The following maps are ratios of the population of the ages 25 and up that have a Bachelor's degree or higher compared to the total number of individuals ages 25 and up in each county in Virginia.


The first map ratio is done using natural breaks as the categorization mode.


The second map ratio is done using pretty breaks as the categorization mode.


The last map ratio is done using equal quantiles as the categorization mode.
